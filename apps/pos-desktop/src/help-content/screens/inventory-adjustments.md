# Ajustes de Inventario

La pantalla de ajustes de inventario permite a los managers corregir diferencias entre el inventario teórico y el real.

## Cuándo usar ajustes de inventario

- **Mermas**: productos dañados, vencidos o robados
- **Sobrantes**: productos encontrados en inventario no registrados
- **Correcciones**: errores de conteo en recepciones
- **Ajustes de lote**: transferencias entre lotes

## Cómo aplicar un ajuste

1. Busca el producto por nombre o código
2. Selecciona el lote específico (si aplica)
3. Ingresa la cantidad (negativa para merma, positiva para sobrante)
4. Selecciona la razón del ajuste
5. Agrega notas adicionales (opcional)
6. Confirma el ajuste

## Razones de ajuste comunes

- Producto dañado
- Producto vencido
- Error de inventario
- Robo / hurto
- Devolución de cliente
- Muestra médica
- Ajuste de calidad

## Offline

Los ajustes se guardan localmente y se sincronizan cuando el servidor está disponible. No necesitas conexión para registrar un ajuste.
