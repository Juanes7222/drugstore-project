# Pantalla de Ventas

La pantalla principal de ventas es donde se registran y cobran los productos. Es la primera pantalla que ves al iniciar sesión.

## Funcionalidades principales

### Agregar productos al carrito

1. Busca el producto por nombre, código interno o código de barras
2. Selecciona el lote correspondiente (se sugiere el que tenga la fecha de vencimiento más próxima — FEFO)
3. Ajusta la cantidad si es necesario
4. El producto se agrega al carrito

### Aplicar descuentos

- Puedes aplicar un descuento porcentual a cada producto
- Los descuentos requieren una razón obligatoria
- Los descuentos por clasificación del cliente se aplican automáticamente

### Gestionar el carrito

- **Modificar cantidad**: usa los botones + / - o escribe la cantidad directamente
- **Eliminar producto**: haz clic en el icono de papelera
- **Vaciar carrito**: usa el botón "Vaciar" en la parte inferior

### Clientes

- Puedes asociar un cliente a la venta para aplicar descuentos por clasificación
- Los clientes frecuentes aparecen en el selector de búsqueda

### Completar la venta

1. Revisa los productos en el carrito
2. Haz clic en "Cobrar" para ir a la pantalla de pago
3. Selecciona el método de pago (efectivo, tarjeta, transferencia, Nequi)
4. Ingresa el monto recibido (para efectivo) o completa la autorización electrónica
5. Confirma la venta

## Atajos de teclado

| Acción       | Atajo        |
|-------------|--------------|
| Nueva venta | `Cmd + N`    |
| Buscar producto | `Escribir` (enfoque automático) |
| Paleta de comandos | `Cmd + K`  |

## Consejos

- Verifica siempre el lote seleccionado antes de agregar al carrito
- Revisa que el cliente tenga el descuento correcto antes de cobrar
- Si la red está caída, la venta se guarda localmente y se sincroniza después
