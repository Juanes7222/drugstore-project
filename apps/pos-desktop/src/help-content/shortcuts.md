# Atajos de Teclado

## Atajos globales

| Acción | Atajo |
|--------|-------|
| Abrir paleta de comandos | `Cmd + K` / `Ctrl + K` |
| Abrir ayuda | `Cmd + /` |
| Ver atajos de teclado | `?` |
| Cerrar panel | `Esc` |
| Ayuda contextual | `F1` |

## Atajos de ventas

| Acción | Atajo |
|--------|-------|
| Nueva venta | `Cmd + N` |
| Reimprimir última factura | `Cmd + Shift + P` |

## Atajos de sincronización

| Acción | Atajo |
|--------|-------|
| Sincronizar ahora | `Cmd + Shift + S` |

## Cómo personalizar atajos

Los managers pueden personalizar los atajos de teclado desde Preferencias:

1. Abre la paleta de comandos (`Cmd + K`)
2. Escribe "Preferencias" y selecciona la opción
3. Ve a la sección "Atajos de teclado"
4. Haz clic en el atajo que quieres cambiar
5. Presiona la nueva combinación de teclas
6. Guarda los cambios

Las personalizaciones son por terminal y no se sincronizan entre equipos.
