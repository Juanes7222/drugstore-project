# Ayuda del Sistema POS

Bienvenido al centro de ayuda del sistema POS de farmacia. Aquí encontrarás guías paso a paso, procedimientos y referencia para todas las funcionalidades del sistema.

## Secciones principales

- **[Procedimientos](/help/procedimientos)** — Guías paso a paso para tareas comunes
- **[Pantallas](/help/pantallas)** — Descripción de cada pantalla del sistema
- **[Atajos de teclado](/help/atajos)** — Lista completa de atajos disponibles

## Soporte offline

Todo el contenido de ayuda está disponible sin conexión a internet. No necesitas estar conectado al servidor para consultar estas guías.

## ¿Necesitas más ayuda?

Si no encuentras lo que buscas, contacta a tu administrador o escribe un término de búsqueda en la paleta de comandos (presiona `Cmd+K` o `Ctrl+K`).
