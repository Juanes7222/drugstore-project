# Cómo cerrar un turno

El cierre de turno es el proceso de finalizar la jornada laboral de un cajero en el sistema.

## Pasos para cerrar un turno

1. Asegúrate de que todas las ventas del turno estén confirmadas
2. Ve a la pantalla de cierre de turno desde el menú de administración
3. Verifica el resumen de ventas del turno:
   - Total de ventas realizadas
   - Total de efectivo recibido
   - Total de pagos electrónicos (tarjeta, transferencia, Nequi)
   - Devoluciones procesadas
4. Realiza el arqueo de caja (conteo físico del efectivo)
5. Ingresa el monto real de efectivo en caja
6. El sistema calcula la diferencia contra el esperado
7. Agrega una nota si hay diferencias significativas
8. Confirma el cierre del turno

## Qué pasa después

- El turno se marca como cerrado (CLOSED)
- Se genera un registro del cierre con las diferencias
- Los datos se sincronizan con el servidor
- El cajero no puede realizar más ventas hasta que se abra un nuevo turno

## Consejos

- Cierra el turno al final de cada jornada
- No dejes turnos abiertos por más de 24 horas
- Si hay diferencias de efectivo, documenta la razón
- El cierre de turno es responsabilidad del manager
