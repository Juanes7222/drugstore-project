# Cómo procesar una devolución

## Paso a paso

### 1. Encuentra la venta original

- Abre la pantalla de devoluciones desde el menú o la paleta de comandos (`Cmd + K`)
- Busca la venta por número local o por cliente
- Selecciona la venta de la lista de resultados

### 2. Selecciona los productos a devolver

- El sistema muestra todos los items de la venta original
- Marca los items que el cliente está devolviendo
- Ajusta la cantidad si solo devuelve parte del producto

### 3. Completa la información de la devolución

- **Razón**: selecciona una razón (producto dañado, incorrecto, etc.)
- **Método de reembolso**: efectivo, tarjeta, o el mismo método de pago original
- **Notas**: cualquier información adicional relevante

### 4. Confirma la devolución

- Revisa el resumen de la devolución
- Confirma para procesarla
- El inventario se actualiza automáticamente
- Si estás offline, la devolución se sincronizará después

## Consideraciones importantes

- Solo se pueden devolver productos de ventas confirmadas
- No se pueden devolver cantidades mayores a las compradas
- Las devoluciones requieren autorización de manager si superan cierto monto
- Los productos controlados tienen reglas especiales de devolución
