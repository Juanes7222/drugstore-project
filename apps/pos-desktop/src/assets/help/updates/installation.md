# Cómo se instala una actualización

Las actualizaciones se instalan de forma automática, pero es importante entender el proceso para saber qué esperar.

## Proceso de instalación

1. **Detección**: El sistema busca actualizaciones periódicamente (cada 6 horas) y al iniciar la app.
2. **Descarga**: La actualización se descarga en segundo plano sin interrumpir tu trabajo.
3. **Respaldo**: Antes de instalar, el sistema hace un respaldo automático de la base de datos.
4. **Instalación**: La instalación ocurre al cerrar la app. Verás una pantalla "Instalando actualización..." por unos segundos.
5. **Migración**: Si la actualización incluye cambios en la base de datos, se aplican automáticamente.
6. **Reinicio**: La app se reinicia con la nueva versión.

## ¿Cuánto tarda?

- **Descarga**: Depende del tamaño y tu conexión. Generalmente 1-5 minutos.
- **Instalación**: 5-15 segundos.
- **Migración**: 10 segundos a 5 minutos, según la cantidad de datos.

## Durante la instalación

- **No cierres la app** ni apagues el equipo mientras veas la pantalla "Instalando actualización...".
- **No desconectes la corriente** durante el proceso.
- Si el proceso se interrumpe, el sistema vuelve automáticamente a la versión anterior.

## Después de la instalación

- Verificá que todo funcione correctamente.
- La primera carga puede ser un poco más lenta mientras se reconstruyen índices.
- Si ves algún comportamiento anómalo, contactá a soporte.
