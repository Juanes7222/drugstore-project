# Qué hacer si una actualización falla

Las actualizaciones están diseñadas para ser seguras, pero ocasionalmente pueden surgir problemas.

## Problemas comunes

### "No se pudo verificar la actualización"

La firma digital de la actualización no coincide. Esto puede ocurrir si:
- El archivo se descargó corrupto.
- Hay un problema de red.
- La clave de firma cambió (en transiciones programadas).

**Qué hacer**: Intentá de nuevo más tarde. Si el problema persiste, contactá a soporte.

### "La descarga se detuvo"

La descarga puede interrumpirse por:
- Pérdida de conexión a internet.
- La app se cerró durante la descarga.

**Qué hacer**: La descarga se reanuda automáticamente cuando haya conexión. No requiere acción.

### "La instalación falló"

La instalación puede fallar por:
- La app se cerró forzosamente durante la instalación.
- El disco no tiene suficiente espacio.
- Un antivirus bloqueó el proceso.

**Qué hacer**: El sistema vuelve automáticamente a la versión anterior. Intentá de nuevo más tarde. Si el problema persiste, contactá a soporte.

### "La migración de datos falló"

La actualización incluye cambios en la base de datos que no pudieron aplicarse.

**Qué hacer**: 
1. El sistema restaura automáticamente el respaldo previo a la actualización.
2. Se vuelve a la versión anterior.
3. Contactá a soporte con el mensaje de error que aparece en pantalla.

### "La app se cierra apenas abre después de una actualización"

El detector de rollback identifica que la nueva versión no es estable.

**Qué hacer**:
1. El sistema vuelve automáticamente a la versión anterior después de 3 intentos fallidos.
2. Se muestra una notificación: "La versión anterior fue restaurada".
3. Contactá a soporte para reportar el problema.

## Contactar a soporte

Si tenés problemas con una actualización:
1. Tomá nota del mensaje de error exacto.
2. Andá a "Acerca de" → "Ver historial" para ver el registro de intentos.
3. Contactá a tu proveedor con esa información.
