# Cómo funciona el plan multi-local

Si tenés un plan multi-local, podés gestionar varias sucursales desde una misma suscripción.

## ¿Cómo funciona?

- Una suscripción puede tener múltiples locales (sucursales).
- Cada local tiene su propia configuración (nombre, dirección, ciudad).
- Cada local tiene su propio límite de puestos de trabajo.
- Los datos de cada local son independientes.

## Agregar un nuevo local

1. El administrador del sistema (tu proveedor) agrega el nuevo local a tu suscripción.
2. Te genera un código de activación para el primer puesto del nuevo local.
3. En el equipo del nuevo local, Actives con ese código.
4. El sistema te va a pedir los datos del local (nombre, dirección, ciudad).

## Gestionar múltiples locales

- En el panel de administración, podés ver todos tus locales.
- Cada local muestra sus puestos activos y su estado.
- Podés generar códigos de activación adicionales para cada local.

## Límites

- Cada plan multi-local tiene un máximo de locales permitidos.
- Cada local tiene un máximo de puestos permitidos.
- Si necesitás más locales o puestos, Contacta a tu proveedor para evaluar un plan superior.

## Ejemplo

Con un plan "Pro Multi-Local 3":
- Podés tener hasta 3 locales
- Cada local puede tener hasta 5 puestos
- Total: hasta 15 puestos en toda la organización
