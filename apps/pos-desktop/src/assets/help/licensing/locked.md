# Qué hacer si tu suscripción está bloqueada

Si ves la pantalla con el mensaje "Suscripción vencida" o "Suscripción bloqueada", no te preocupes. Seguí estos pasos:

## Podés seguir haciendo esto

- ✅ Ver todas tus ventas, clientes y productos
- ✅ Exportar tus datos a CSV/JSON
- ✅ Ver报告es e histórico
- ✅ Imprimir comprobantes de operaciones existentes

## No podés hacer esto hasta renovar

- ❌ Crear nuevas ventas
- ❌ Hacer devoluciones
- ❌ Ajustar inventario
- ❌ Abrir o cerrar turnos
- ❌ Crear recetas

## Pasos para resolverlo

1. **Contacta a tu proveedor** para regularizar el pago.
2. **El proveedor registra el pago** y reactiva la suscripción.
3. **El sistema se actualiza automáticamente** en el próximo check-in (cada 24 horas).
4. **También podés forzar la actualización** desde la pantalla de estado de licencia → "Renovar ahora".
5. Una vez procesado, el sistema vuelve a la normalidad. **No perdés ningún dato.**

## Exportar tus datos

Siempre podés exportar tus datos, incluso si el sistema está bloqueado:
1. Andá a la pantalla de estado de licencia.
2. Hacé clic en "Exportar datos".
3. Elegí el formato (CSV o JSON).
4. Los datos se descargan a tu equipo.
