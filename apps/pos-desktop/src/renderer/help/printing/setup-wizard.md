# Asistente de configuración de impresoras

El asistente te guía paso a paso para configurar tus impresoras y dispositivos periféricos. No necesitas conocimientos técnicos.

## Para empezar

1. Abre el asistente desde el menú de administración o escribiendo "Configurar impresoras" en la paleta de comandos (F1 o Ctrl+K)
2. Haz clic en "Empezar" para iniciar el proceso
3. El asistente buscará automáticamente todas las impresoras conectadas a la estación de trabajo

## Pasos del asistente

1. **Bienvenida** — Explicación del proceso (3-5 minutos)
2. **Detección** — Búsqueda de impresoras y dispositivos conectados
3. **Dispositivos encontrados** — Revisa y nombra cada impresora
4. **Asignación de trabajos** — Elige qué tipo de documentos imprime cada impresora
5. **Pruebas** — Verifica que cada impresora funciona correctamente
6. **Configuración de respaldo** — Decide qué hacer si una impresora falla
7. **Resumen** — Revisa y guarda la configuración

## Consejos

- Conecta todas las impresoras antes de iniciar el asistente
- Usa nombres descriptivos como "Impresora de recibos del mostrador" o "Impresora de facturas"
- Siempre ejecuta la prueba de impresión para verificar la conexión
- Si no encuentras una impresora de red, activa la "Búsqueda en la red local" en el paso 2
